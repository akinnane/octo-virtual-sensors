#  Octo Virtual Sensors
 
 Update the virtual sensors on an Aqua computer Octo

 Usage:
 ```
 use octo_virtual_sensors;
 let mut octo = Octo::new().unwrap();
 octo.update_virtual_sensors(&[1, 2, 3]).unwrap();
 ```
